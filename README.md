Программа для работы и анализа SQL баз данных СибФТИ СФНЦА РАН и предсказаний оптимальных параметров посадки растений на экспериментальных полях института. Реализован графический интерфейс для баз данных, функции формирования отчетов по базам данных, создание выборок по параметрам и графический интерфейс представления графических данных по выборке (графики и фотографии).
